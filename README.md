# hololive-seihousou
hololive直播及直播预定列表查询工具  
## 注意！因为底层为Selenium模拟实现爬虫，所以实际性能较低，慎用！  
  
### 主要依赖  
* [PHP](https://www.php.net/)  
* [Python](https://www.python.org/)  
* [Python Selenium](https://pypi.org/project/selenium/)  
* [Python Beautifulsoup4](https://pypi.org/project/beautifulsoup4/)  
* [Python lxml](https://pypi.org/project/beautifulsoup4/)  
* [火狐](https://www.firefox.com.cn/)  
* [火狐geckodriver](https://github.com/mozilla/geckodriver/releases/)
